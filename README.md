# @coti/pod-sdk

SDK for building privacy dApps on EVM with COTI Privacy on Demand (PoD): Solidity contracts, async Inbox messaging patterns, and TypeScript crypto helpers.

## Install

```bash
npm install @coti/pod-sdk
```
